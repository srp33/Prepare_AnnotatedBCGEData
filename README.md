# Curated Compendium of Breast Cancer Gene-Expression Datasets

This repository contains the code used to prepare the datasets for [this collection](https://zenodo.org/communities/annnotatedbcgedata/about).

It is assumed that the user has Docker installed. Installation steps for Docker can be found at [here](https://docs.docker.com/get-docker/).

Each step should be run in sequence (i.e., 1_ should be run first, followed by 2_, followed by 3_*, etc.).

To run each step, your working directory should be the directory for that step.

Each step is run by executing the `run.sh` script for that step.
